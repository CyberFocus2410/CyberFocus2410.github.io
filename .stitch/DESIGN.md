# Stitch Design System — Vivan Mittal Portfolio

## 🎨 Visual Identity
- **Atmosphere**: Dark terminal-meets-editorial. Atmospheric, premium, and cybersecurity-flavored.
- **Mood**: Professional, Handcrafted, Technical, Intentional.

## 🌈 Color Palette
### Dark Theme (Default)
- **Background Base**: `#020817` (Deep Navy)
- **Background Deep**: `#010610` (Midnight)
- **Surface**: `#0d1424` (Glassy cards)
- **Accent Primary**: `#38bdf8` (Electric Sky Blue)
- **Accent Secondary**: `#a855f7` (Vivid Purple)
- **Accent Tertiary**: `#f59e0b` (Amber highlights)
- **Text Primary**: `#f8fafc` (Off-white)
- **Text Muted**: `#94a3b8` (Slate blue)
- **Border Subhuman**: `rgba(255, 255, 255, 0.05)` (Ultra-subtle)

### Light Theme
- **Background Base**: `#f8fafc`
- **Surface**: `rgba(255, 255, 255, 0.8)`
- **Accent**: `#4f46e5` (Deep Indigo)

## ✍️ Typography
- **Display (Hero)**: `Syne` (Bold, geometric, 800 weight)
- **Body**: `DM Sans` (Clean, modern, 400-600 weight)
- **Monospace**: `JetBrains Mono` (Technical, used for labels and code)

## 📏 Layout & Structure
- **Bento Grid**: Used for Projects and Skill displays.
- **Section Spacing**: Large vertical paddings (`padding-top: 120px`) for breathability.
- **Glassmorphism**: `backdrop-filter: blur(16px)` with `border: 1px solid rgba(255,255,255,0.08)`.

## ✨ Micro-interactions
- **Custom Cursor**: A dot with a following ring that reacts to interactive elements.
- **Matrix Rain**: Subtle background overlay (10% opacity) for cybersecurity vibe.
- **Hover States**: Subtle lift (`translateY(-4px)`), increased glow, and border illumination.
- **Scanlines**: Subtle horizontal line animation for terminal feel.
